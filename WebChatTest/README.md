# WebChatTest

This is a client for test WebChat applications.

# Prerequisites

* Java 8
* Maven 3.0.5
* node v6.9.0
* npm v4.1.2
* tsc v2.2.1

## SetUp

```sh
$ cd TFG-WebChat/WebChatTest
$ mvn install
$ cd src/main/resources/webroot
$ npm install
$ tsc
```

# Run test client

```sh
$ cd WebChatTest
$ mvn test
```
